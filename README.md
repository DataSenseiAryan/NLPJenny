## Dependencies
- Python3
- Streamlit
- Othes libraries (Everything in Pipfile and Requirement.txt)

## How to RUN:
- Run 
   > git clone https://github.com/aryanc55/NLPJenny.git
   
   > cd NLPJenny
   
   > pip install -r requirement.txt
   
   > streamlit run NLPJenny.py
